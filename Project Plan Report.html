Mission

Purpose of website

Townsville Community Music Centre's website is established for two purposes as per content provided by client -

    Membership
  • To recruit new members
  • To increase engagement of members in our activities, e.g. volunteering for administration, organisation, and promotion of concerts and workshops
    Concerts
  • To increase ticket sales, especially to the general public
  • To encourage performers to let us organise concerts for them
  • To encourage performers to cooperate with promotional materials, photo-shoots, media releases, etc.

Testing Details

Each member was assigned to do the following in order to achieve the designed website for the client:

Objective:

Our team conducted usability testing to users ranging from 17 - 50 years old. We believe that these age represents aspiring solo artists, musical bands, choirs,
and musical performers alike.

Our team conducted the following tests:

  1. Functionality Testing
  2. Usability testing
  3. Interface testing
  4. Compatibility testing
  5. Performance testing

Userbility Testing Checklist and User Ratings

User Comments on Website
  • Website functionality is overall great.
  • Everything is interactive and nice.
  • Script just needs to have windows outside close button on the forms.
  • Make the forms more interactive by adding icons on the input fields to guide users what they completed or missed out.
  • Well-done. Everything is done professionally.
Conclusion on Userbility Testing
  • Website was modified again to make it more interactive.
  • Navigation (scroll bars) are improved to have a smooth scrolling effect.
  • Icons were modified to make the User Interface more consistent.
  • Sign-up forms were modified to make it more interactive.
